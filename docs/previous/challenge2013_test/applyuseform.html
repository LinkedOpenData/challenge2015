メールでのご応募について

現在、応募システムの障害により、Webサイトでの受付を停止しております。応募者の皆さま、並びに関係者の皆さまにはご迷惑をおかけし、大変申し訳ございません。

システム復旧までの間、下記の通りメールでのご応募とさせていただきます。

下記リンクから、各部門の応募シート(WORDファイル)をダウンロードしてください。
必要事項を記入の上、下記の通りメールでご応募ください。

宛先：lod-challenge[at]sfc.keio.ac.jp ([at]を@に置き換えてください)

件名：応募部門名(アイデア部門など)

添付ファイル：応募フォーム

なお、応募頂きました作品情報は、システム復旧後に実行委員会で登録するとともに、登録後に応募者の皆様へご連絡致します。

何卒よろしくお願い申し上げます。
